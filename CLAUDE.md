# My First App

I am building an app that lets you practice texting in Spanish with an AI tutor.

## Tech Stack
- Next.js 14 with App Router
- Typescript
- Tailwind CSS

## Project Structure
- src/app/ - Pages and API routes
- src/components/ - Reusable UI components
- src/components/ui/ - Small generic components (Button, Card, Input)
- src/lib/ - Utility functions and service connections
- src/types/ - TypeScript types

## Conventions
- Use Tailwind for all styling
- Components use PascalCase (MyComponent.tsx)
- One component per file
- Use TypeScript for everything
